ConvertIQ ML Workflow (Final Version)



1️⃣ Collect visitor behaviour using SalesIQ tracking

2️⃣ Bot extracts key features (time on page, idle time, sentiment, etc.)

3️⃣ Bot sends these features to the ML API

4️⃣ API loads the RandomForest model

5️⃣ Model predicts drop-off risk (Low/Medium/High)

6️⃣ API returns prediction + confidence

7️⃣ SalesIQ bot decides action:



High → operator alert + proactive help



Medium → show suggestions



Low → normal interaction

8️⃣ Analytics widget shows risk trends
